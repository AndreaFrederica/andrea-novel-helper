本次安装内容：
------------------------------------------------------------
• NovelHelper 主程序
  安装路径：{app}

• Git
  如果系统中未检测到 Git，则已自动安装到：
  C:\Program Files\Git

• Visual Studio Code (User Setup)
  如果系统中未检测到 VS Code，则已自动安装到：
  %LOCALAPPDATA%\Programs\Microsoft VS Code

• VS Code 扩展
  - 离线扩展：安装自本地 vsix 目录（若存在）
  - 在线扩展：从 extensions.txt 补充安装
  - 中文语言包：已自动安装

• VS Code 配置
  - 已写入默认 settings.json 模板
  - 界面语言已设置为简体中文

后续操作：
------------------------------------------------------------
1. 在桌面或开始菜单中使用快捷方式启动 VS Code。
2. 您可以在 VS Code 中继续安装其他扩展。
3. 如果需要调整 Git 用户信息，请在命令行执行：
     git config --global user.name  "你的名字"
     git config --global user.email "你的邮箱"

常见问题：
------------------------------------------------------------
- 如果启动时提示找不到 Git 或 Code，请确认安装目录
  已正确加入系统环境变量，或重新运行安装程序。
- 如果扩展安装不完整，可手动运行：
     code --install-extension <扩展ID>
